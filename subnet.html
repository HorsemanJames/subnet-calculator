Formula for Network ID:

NETWORK_ID = IP_ADDRESS & SUBNET_MASK

Formula for Broadcast Address:

BROADCAST_ADDRESS = IP_ADRESS | (~NETWORK_ID)

Explanation

Understanding of IPv4 subnet calculations is useful if you ever need to calculate the broadcast address for a subnet or if you want to understand how routing decisions are made. The broadcast address is the address to which you would send a packet if you wanted to send that packet to all IP addresses in your subnet. To understand how routing decisions are made you need to calculate the network ID.

When an operating system is trying to determine where to send an IP packet, the operating system will consult the routing table. Each routing table entry contains a network ID, subnet mask, and next hop. The operating system will look at the destination IP address of the packet and construct a network ID based on the subnet masks of the various entries in the routing table. If the resulting network ID matches the network ID in the routing table entry, then the operating system will route the IP packet to the next hop listed in the routing table entry.
